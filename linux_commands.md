# Useful Linux Commands
## ls - list files   
#### list all files with all attributes human readable

```
ls -hal
```
#### list file inodes
```
ls -i
```
#### colored folders and files

```
ls --color
```

## pwd - print work directory
#### value of current working
```
pwd -L
```

#### resolves symbolic links
```
pwd -P
```
    

## cd - change directory

```
cd -P
```

