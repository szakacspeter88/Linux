# Useful Linux Commands
## ls - list files   
#### list all files with all attributes

```
ls -all
```
#### list file inodes
```
ls -i
```
#### list file inodes
```
ls -h
```
#### human readable

```
ls --color
```
#### colored folders and files

## pwd - print work directory
#### value of current working
```
pwd -L
```

#### resolves symbolic links
```
pwd -P
```
